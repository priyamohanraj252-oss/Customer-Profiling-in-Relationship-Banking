# 📊 Customer Profile Analysis in Relational Banking

## 📌 Overview
This project focuses on analyzing customer data in the banking domain to understand customer behavior, identify patterns, and generate actionable insights to improve customer relationships and business decision-making.

---

## 🎯 Objectives
- Analyze customer data to identify trends and patterns  
- Perform customer segmentation  
- Generate insights for better customer relationship management  
- Create interactive dashboards for visualization  

---

## 🛠️ Tools & Technologies
- SQL  
- Microsoft Excel  
- Power BI  

---

## 📂 Project Workflow

### 1. Data Collection
- Collected customer-related dataset for analysis  

### 2. Data Cleaning & Preprocessing
- Handled missing values  
- Removed inconsistencies  
- Structured the dataset for analysis  

### 3. Data Analysis
- Used SQL for querying and analysis  
- Identified key trends and customer behavior patterns  

### 4. Data Visualization
- Built interactive dashboards using Power BI  
- Visualized insights using charts and graphs  

---

## 📊 Key Insights
- Identified different customer segments  
- Analyzed customer behavior patterns  
- Provided insights to improve customer engagement and retention

---

## 🙌 Conclusion
This project helped in understanding real-world data analysis processes, including data cleaning, analysis, and visualization, to support business decisions.
